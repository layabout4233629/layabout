### 。

python是一个非常适用于个人学习的编程语言，有几个优势：
1 优美简洁的语法（从C++看到python后有一种相见恨晚的感觉，特别是对有代码洁癖的我而言）
2 非常多好用的库，并且只需要一句import便可以使用
3 无需编译修改方便，随改随用

缺点有一个：
1 环境迁移不方便（虽然已经有了很多解决方案）
2 代码没有静态检查
